## SECTION 7: STABLE PROTECTION HOOK UI

When a pool uses the Stable Protection Hook, the Liquidity modal must show additional information:

- **Zone Badge**: Color-coded indicator (Green=HEALTHY, Yellow=MINOR, Orange=MODERATE, Red=SEVERE, Black=CRITICAL)
- **Dynamic Fee**: Current fee in basis points with direction indicator
- **Fee Schedule**:
  - Toward peg: base fee × 0.5
  - Away from peg: base fee × 3.0
- **Circuit Breaker Status**: Active/Inactive
- **Zone Thresholds**: 0.1% / 0.5% / 2.0% / 5.0%

Read this data from the hook's view functions:
- `getPoolState(poolId)` → returns current zone + last deviation
- `getPoolConfig(poolId)` → returns fee config + thresholds
